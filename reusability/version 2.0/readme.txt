Se ejecuta como un plugin de eclipse.
Generar el jar y copiarlo en el directorio de plugins.
Aparecera una nueva opcion llamada Artist a traves de la que se ejecutara el plugin.

Inputs de entrada: 
config.xml: fichero en el que se establecen los criterios a partir de los cuales se determina si un metodo es reusable o no.
ruta donde se encuentra el proyecto legacy.
